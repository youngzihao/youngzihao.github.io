创建编译头文件 PCH ( Pre-Compiled Header)

1.创建 PCH 文件

Command+N -> iOS -> other -> PCH file -> PrefixHeader.pch

2.添加 PCH 文件路径

将 Building Setting 中的 Prefix Header 选项的路径添加为“$(SRCROOT)/项目名称/pch文件名”
例如：$(SRCROOT)/PrefixHeader.pch
$(SRCROOT) 的意思就是工程根目录的意思

3.将 Precompile Prefix Header 为 YES

预编译后 PCH 文件会被缓存起来，可以提高编译速度
