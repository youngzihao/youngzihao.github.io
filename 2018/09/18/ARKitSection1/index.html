Session Management

Create the configuration

  1. 创建一个 ARWorldTrackingConfiguration 实例，分配给了 config 。
  2. 然后设置了 worldAlignment 的属性，也就是具体说明虚拟的内容将如何与现实世界相关联。
  3. 不允许 AR session 捕捉声音，还存在一个属性 isLightEstimationEnabled 之后会学到。
  4. 运行 AR session ，并使用你刚刚设置好的配置，现在实时追踪 6DOF 维度的数据。

Controll an AR session

  1. Pause:
    ARSession.pause()
  2. Resume:
    ARSession.run()
  3. Update:
    ARSession.run(ARSessionConfig)
  4. Reset:
    ARSession.run(_:options:)
